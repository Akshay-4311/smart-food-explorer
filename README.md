# Smart Food Explorer

## 📌 Project Description
Smart Food Explorer is a simple web application that allows users to search for food recipes using a public API. Users can view meal details, filter results by category, and sort them alphabetically.

## 🚀 Features
- Search meals using API
- Filter meals by category
- Sort meals (A–Z, Z–A)
- View detailed recipe information
- Dark mode toggle
- Responsive design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- MealDB API

## 🌐 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser

## 📡 API Used
https://www.themealdb.com/api.php
